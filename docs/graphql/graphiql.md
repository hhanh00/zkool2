---
title: GraphIQL
---

`GraphIQL` is an interactive dev/test page embedded in `zkool_graphql`. It is a
convenient way to try out the GraphQL API without using a client, *directly*
from the web browser.

Run `zkool_graphql` and open the page at
[http://localhost:8000/graphiql](http://localhost:8000/graphiql)

![GraphIQL](images/01.graphiql.png)

> Examples from this documentation use the GraphIQL tool.
