Managing Categories

Zkool comes with a list of predefined categories, but you can make your own list.

Open the category editor from the account manager page.

The List of Categories shows up. Like the Folder Manager, add new categories with the "+" button on the app bar.

Add, Edit and Delete categories by first selecting them and using the associated button on the app bar.
